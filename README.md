 Password Security Analyzer (Python)

A simple yet powerful cybersecurity project built with Python that evaluates password strength, estimates entropy, and generates secure passwords.

 Features
- Password strength checker (score-based system)
- Entropy calculation for security analysis
- Password generator with strong randomness
- Real-time feedback to improve weak passwords

 What I Learned
- Basics of cybersecurity and password security
- Entropy concept in information security
- Regular expressions in Python
- Secure password generation logic

Tech Stack
- Python
- Regex
- Math module
- Random module

 Purpose
This project was built for learning cybersecurity fundamentals and improving secure coding practices.
